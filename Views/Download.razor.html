GacUI for Windows

Go to https://github.com/vczh-libraries/Release, there are 3 folders:

  • Import : GacUI release source code. See the tutorial for details.
  • Tools : Code generation tools for developing GacUI projects.
  • Tutorial : Sample code showing all the features that GacUI has provided.

GacUI for OSX

The OSX version is almost done, but the release has not been integrated into the GacUI release repo yet. Please go to https://github.com/vczh-libraries/iGac for the source code.

GacUI for Linux

The Linux version is under development. Please go to https://github.com/vczh-libraries/XGac if you want to have a try.

Source Code

The source code in GacUI release is compacted to make the development of GacUI projects easier. If you want to study from the source code, you will find it is hard to read a C++ file with tens of thousands of lines. In this case, you can go to https://github.com/vczh-libraries, there are different repos for GacUI's sub projects, some sub projects have unit tests.

  • Vlpp : Cross platform C++ libraries for general purposes.
  • Workflow : A script language that is heavily used in GacUI, which has a good interoperability with C++. C++ classes can be registered by a reflection system (from Vlpp), and Workflow script code can call them directly.
  • GacUI : GacUI source code. All GUI classes are reflectable.
  • iGac : GacUI for OSX
  • XGac : GacUI for Linux
  • GacJS : GacUI on Browsers! This project is far from finish.
  • vczh-libraries.github.io : The source code of this website.
  • Tools : Command line tools for this organization.
