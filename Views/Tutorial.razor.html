Any GacUI project is highly recommended to create using the XML resource. You can refer to the CppXml demo. Generally a GacUI project can be created by the following steps:

  • Create a VC++ project using Visual Studio 2013 or higher version, and add all source codes from the Release folder to your project.
  • Prepare your XML resource.
  • Generate necessary C++ classes and the binary resource file from your XML resource using GacGen.exe.
  • Complete the WinMain and GuiMain functions, load the binary resource file, create the main window and run.

Notes : The binary resource file don't necessarly be a file. You can embed the file as a resource linking to your exe. After you load it in the memory, you can use MemoryStream instead of FileStream to load the resource.
