Source code of all demos

Source code of all demos are in the "Tutorial" folder of the GacUI for Windows repo:

  • Hello, world! : Every possible ways to create a hello world GacUI project.
    • Cpp : Create a GacUI project purely in C++.
    • WorkflowScript : Still a C++ project but use script to create the window.
    • CppXml : Create a GacUI project from an XML resource, with some C++ code generated from it.
    • MVVM : Using the MVVM pattern with XML resource.
  • Layout : Demonstrates all layout features.
    • Alignment : Align controls.
    • Stack : Stack layout.
    • Flow : Flow layout with configurable axis directions.
    • Table : Table layout.
    • RichTextEmbedding : Embed controls in a rich text document.
  • Controls : Demonstrates all controls and control templates. This demo is under development.
  • XML : Demonstrates all features in the XML resource file. This demo is under development.
  • etc ...

Running a demo

Using Visual Studio 2013 or higher version to open the solution, and then run the demo by pressing F5.

If you want to modify the demo for practicing, you may notice that there are a Codegen.bat file in the UI folder in every demo projects. Modify the xml and run the bat file, the corresponding binary resource in the UIRes folder will be updated. After these steps are done, press F5 to see the result!
